# 🔥 Jogo do Termo em C#
![alt text](JogoTermo_5CNY6VvcGb.gif)

Um clone do popular jogo de adivinhação de palavras (Wordle/Termo) desenvolvido em C# como aplicação de console. O projeto foca em lógica de programação, manipulação de arrays e interação via terminal.

## 🚀 Como Funciona
O jogo escolhe uma palavra aleatória de 5 letras, e você tem 6 tentativas para adivinhar:
* 🟩 **Verde**: A letra está correta e na posição certa.
* 🟨 **Amarelo**: A letra existe na palavra, mas está na posição errada.
* 🔴 **Vermelho**: A letra não faz parte da palavra.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** C# (.NET)
* **Namespace:** `System.Security.Cryptography` (para geração de números aleatórios seguros)
